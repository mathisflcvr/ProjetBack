# Projet Node Back JS

Le concept : Ce jeu est un SHMUP sur la base de Node JS, CSS, HTML, Express JS.
Le jeu se lance avec localhost:3000, il se joue grâce au flèches directionnelles pour se déplacer, et la barre d'espace sert à tirer.
Dans Gentil Shmup, tu incarnes le personnage "Llovni", un gogole dans une soucoupe volante qui s'est perdu sur l'autoroute de l'espace.
Le but du jeu est d'avancer au travers les différents niveaux, d'aller le plus loin possible en shootant un maximum d'ennemis et en esquivant le plus d'obstacles.
À chaque partie, tu disposes de 3 vies, un dégât équivaut à une vie en moins. Quand vous n'avez plus de vie, la partie se termine. GAME OVER !
Une fois de retour sur l'écran d'acceuil, vous pouvez appuyer sur la touche (Espace) pour relancer une partie, ou appuyer sur la touche (Sup) pour quitter le jeu
